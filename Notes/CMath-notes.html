Sets - Exercise 1.1 Solutions

Problem 1

Part (a): Present the cardinality of sets with examples

Present the cardinality of sets with examples and show it to your teacher.

Solution:

Cardinality of a Set: The cardinality of a set is the number of elements in that set. It is denoted by \( n(A) \) or \( |A| \).

Examples:

  • Set A = {1, 2, 3, 4, 5}
    Cardinality: \( n(A) = 5 \)
  • Set B = {a, b, c}
    Cardinality: \( n(B) = 3 \)
  • Set C = {red, blue, green, yellow}
    Cardinality: \( n(C) = 4 \)
  • Empty Set ∅ = { }
    Cardinality: \( n(∅) = 0 \)
  • Set D = {x | x is a vowel in English}
    D = {a, e, i, o, u}, so \( n(D) = 5 \)

Part (b): For two sets A and B, A ⊂ B, find the values of n(A∪B) and n(A∩B)

Solution:

When A ⊂ B (A is a subset of B), this means every element of A is also in B.

Key Properties when A ⊂ B:

\[ A \cap B = A \text{ (since all elements of A are in B)} \] \[ A \cup B = B \text{ (since A is completely contained in B)} \]

Therefore:

  • \( n(A \cup B) = n(B) \)
  • \( n(A \cap B) = n(A) \)

Example:

Let A = {1, 2, 3} and B = {1, 2, 3, 4, 5, 6}

Since A ⊂ B:

  • \( A \cap B = \{1, 2, 3\} = A \), so \( n(A \cap B) = 3 = n(A) \)
  • \( A \cup B = \{1, 2, 3, 4, 5, 6\} = B \), so \( n(A \cup B) = 6 = n(B) \)

Part (c): If A and B are overlapping sets, state the formula for n(A∪B)

Solution:

When two sets A and B are overlapping (they have some common elements), we use the inclusion-exclusion principle.

\[ n(A \cup B) = n(A) + n(B) - n(A \cap B) \]

Alternative form:

\[ n(A \cup B) = n^o(A) + n^o(B) + n(A \cap B) \]

where \( n^o(A) \) represents elements only in A, and \( n^o(B) \) represents elements only in B.

Example:

Let A = {1, 2, 3, 4} and B = {3, 4, 5, 6}

  • \( n(A) = 4 \)
  • \( n(B) = 4 \)
  • \( A \cap B = \{3, 4\} \), so \( n(A \cap B) = 2 \)
  • \( A \cup B = \{1, 2, 3, 4, 5, 6\} \)

Using the formula: \( n(A \cup B) = 4 + 4 - 2 = 6 \) ✓

Part (d): There are 12 and 8 elements in sets A and B respectively. Find the minimum number of elements in n(A∪B)

Solution:

Given: \( n(A) = 12 \) and \( n(B) = 8 \)

Using the formula: \( n(A \cup B) = n(A) + n(B) - n(A \cap B) \)

\[ n(A \cup B) = 12 + 8 - n(A \cap B) = 20 - n(A \cap B) \]

To minimize \( n(A \cup B) \), we need to maximize \( n(A \cap B) \).

The maximum value of \( n(A \cap B) \) is limited by the smaller set, so:

\[ n(A \cap B) \leq \min(n(A), n(B)) = \min(12, 8) = 8 \]

Therefore, the minimum value of \( n(A \cup B) \) is:

\[ n(A \cup B)_{\min} = 20 - 8 = 12 \]

Answer: The minimum number of elements in A∪B is 12.

Explanation:

This occurs when set B is completely contained within set A (B ⊂ A), meaning all 8 elements of B are also in A.

Problem 2

In the given Venn-diagram, 80 people are in set M, 90 people are in set E and 15 people are not in both the sets. The diagram shows values 20, 60, 30, 15.

Determine the cardinality of following sets:

  1. \( n^o(M) \)
  2. \( n^o(E) \)
  3. \( n(M) \)
  4. \( n(E) \)
  5. \( n(M \cup E) \)
  6. \( n(M \cap E) \)
  7. \( n(\overline{M \cup E}) \)
  8. \( n(U) \)

Solution:

From the Venn diagram, we can identify:

  • Only in M: 20 people
  • In both M and E: 60 people
  • Only in E: 30 people
  • In neither M nor E: 15 people
  1. \( n^o(M) = 20 \) (people only in M)
  2. \( n^o(E) = 30 \) (people only in E)
  3. \( n(M) = 20 + 60 = 80 \) (total in M)
  4. \( n(E) = 30 + 60 = 90 \) (total in E)
  5. \( n(M \cup E) = 20 + 60 + 30 = 110 \) (in M or E or both)
  6. \( n(M \cap E) = 60 \) (in both M and E)
  7. \( n(\overline{M \cup E}) = 15 \) (not in M or E)
  8. \( n(U) = 20 + 60 + 30 + 15 = 125 \) (total universal set)
